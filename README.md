# Smart Campus Information System

## Project Description

The Smart Campus Information System is a Python-based mini project developed by integrating all modules from Lab 1 to Lab 8.

The project helps manage:

* Student registration
* Course enrollment
* Student academic records
* Fee calculation
* Sorting and searching student data
* File handling
* Directory scanning
* Student performance analytics

---

# Features

* Student Registration and Grade Evaluation
* Course Enrollment Management
* Student Record Management
* Searching and Sorting Student IDs
* Fee Calculation using Functions
* File Handling for Academic Records
* Directory Scanning with Exception Handling
* Student Performance Analytics using NumPy, Pandas, and Matplotlib

---
